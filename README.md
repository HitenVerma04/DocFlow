# 🏥 DocFlow — Hospital Management System

A full-stack Hospital Management System built with **Spring Boot**, **React**, and **PostgreSQL**.

## 🔧 Tech Stack

**Backend:** Java, Spring Boot, Spring Data JPA, Hibernate, REST APIs  
**Frontend:** React, Tailwind CSS  
**Database:** PostgreSQL  
**Security:** JWT Authentication, Role-Based Access Control  
**Tools:** Docker, Swagger, Postman  

## 🚀 Features (In Progress)

- [ ] Patient Management (CRUD)
- [ ] Doctor Management
- [ ] Appointment Booking System
- [ ] JWT Authentication (Login/Register)
- [ ] Role-Based Access (Admin, Doctor, Patient)
- [ ] Search & Pagination
- [ ] Swagger API Documentation
- [ ] Docker Containerization
